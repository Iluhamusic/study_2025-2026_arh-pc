### 1. Цель работы

Проверить работу виртуализации и определить, обнаруживает ли операционная система гипервизор.

### 2. Среда выполнения

Хост-система: Windows  
Гипервизор: Oracle VirtualBox  
Гостевая ОС: Fedora Linux (Sway)

### 3. Ход работы

В терминале была выполнена команда:

```bash
sudo dmesg | grep -i hypervisor
```

### 4. Результат выполнения

В результате выполнения команды получена строка:

```
Hypervisor detected: KVM
```

Это подтверждает, что операционная система запущена внутри виртуальной машины.

### 5. Вывод

Виртуализация работает корректно. Fedora успешно функционирует в среде Oracle VirtualBox.
