# Лабораторная работа №X
## Тема: Работа с правами доступа в Linux

## Цель работы
Изучить команды chmod, cp, mv и работу с правами доступа.

## Ход работы

### 1. Копирование файла
cp feathers file.old

### 2. Изменение прав доступа
chmod u-r feathers

Попытка чтения:
cat feathers
→ отказано в доступе

### 3. Восстановление прав
chmod u+r feathers

### 4. Работа с каталогами
chmod u-x play
cd play
→ отказано в доступе

chmod u+x play

### 5. Копирование и перемещение
cp -r play fun
mv fun play/games

## Вывод
В ходе работы были изучены команды управления правами доступа и операции с файлами и каталогами.
