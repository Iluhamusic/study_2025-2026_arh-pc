# Отчёт по лабораторной работе №2

## Цель работы

Изучить систему контроля версий Git, настроить пользовательские параметры, создать SSH-ключи и выполнить клонирование удалённого репозитория.

## Задание

Выполнить базовые операции работы с Git:

- установить имя и email пользователя
- создать SSH-ключ
- добавить ключ в систему
- клонировать удалённый репозиторий

## Ход выполнения работы

Настройка имени пользователя Git:

git config --global user.name "Ilya"
Настройка электронной почты:

git config --global user.email "example@email.com"
Проверка настроек Git:

git config --list
Создание SSH-ключа:

ssh-keygen -t ed25519 -C "example@email.com"
Просмотр созданного ключа:

cat ~/.ssh/id_ed25519.pub
Клонирование удалённого репозитория:

git clone git@github.com:user/repository.git
Переход в каталог репозитория:

cd repository
Проверка состояния репозитория:

git status
