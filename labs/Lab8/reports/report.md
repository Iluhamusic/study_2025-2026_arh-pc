## Ответы на контрольные вопросы  

1. stdin, stdout, stderr  
2. > — перезапись, >> — добавление  
3. | — передаёт вывод  
4. Процесс — выполняемая программа  
5. PID — ID процесса, GID — ID группы  
6. Jobs — фоновые задачи  
7. top/htop — мониторинг процессов  
8. find — поиск файлов  
9. grep — поиск по содержимому  
10. df — диск  
11. du — размер  
12. kill PID — завершение процесса
