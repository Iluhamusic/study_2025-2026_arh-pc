## Задание 1

Используя команду getopts и grep, был создан командный файл для анализа аргументов командной строки.

### Листинг программы task1.sh

#!/bin/bash

ignore_case=""
line_numbers=""

while getopts "i:o:p:cn" opt; do
    case $opt in
        i) input="$OPTARG" ;;
        o) output="$OPTARG" ;;
        p) pattern="$OPTARG" ;;
        c) ignore_case="-i" ;;
        n) line_numbers="-n" ;;
        *) echo "Ошибка"; exit 1 ;;
    esac
done

grep $ignore_case $line_numbers "$pattern" "$input" > "$output"

echo "Поиск завершён"

## Задание 3

Был создан командный файл для автоматического создания и удаления файлов.

### Листинг программы `task3.sh`

bash
#!/bin/bash

N=$1

for ((i=1; i<=N; i++))
do
    touch "$i.tmp"
done

echo "Создано $N файлов"

ls *.tmp

rm -f *.tmp

echo "Файлы удалены"

# Ответы на контрольные вопросы

## 1. Каково предназначение команды getopts?

Команда `getopts` используется для разбора аргументов командной строки в shell-скриптах.

---

## 2. Какое отношение метасимволы имеют к генерации имён файлов?

Метасимволы (`*`, `?`, `[]`) используются для автоматического выбора и генерации имён файлов.

---

## 3. Какие операторы управления действиями вы знаете?

Операторы управления:

- `if`
- `case`

---

## 4. Какие операторы используются для прерывания цикла?

Для прерывания циклов используются:

- `break`
- `continue`

---

## 5. Для чего нужны команды false и true?

Команда `true` всегда возвращает код завершения 0, а `false` — ненулевой код ошибки.

---

## 6. Что означает строка if test -f man$f/$i.$s?

Данная строка проверяет существование файла.

bash
if test -f man$f/$i.$s
`

Если файл существует, условие считается истинным.
