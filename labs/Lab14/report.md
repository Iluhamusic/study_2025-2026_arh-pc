## Задание 1. Упрощённый механизм семафоров

Был создан командный файл semaphore.sh, реализующий механизм блокировки ресурса с помощью lock-файла.

### Листинг программы

#!/bin/bash

LOCKFILE="/tmp/mylock"

while [ -f "$LOCKFILE" ]
do
    echo "Ресурс занят. Ожидание..."
    sleep 2
done

touch "$LOCKFILE"

echo "Ресурс получен процессом $$"

sleep 10

echo "Освобождение ресурса"

rm -f "$LOCKFILE"

### Результат выполнения

Программа ожидает освобождения ресурса при наличии lock-файла. После освобождения ресурс захватывается процессом, затем освобождается спустя некоторое время.

## Задание 2. Реализация команды man

Был создан командный файл myman.sh, отображающий справку по команде из каталога /usr/share/man/man1.

### Листинг программы

#!/bin/bash

FILE="/usr/share/man/man1/$1.1.gz"

if [ -f "$FILE" ]
then
    gzip -dc "$FILE" | less
else
    echo "Справка для команды $1 не найдена"
fi

### Результат выполнения

При запуске программы с аргументом командной строки отображается справочная информация соответствующей команды.

Пример:

./myman.sh ls

## Задание 3. Генерация случайной последовательности букв

Был создан командный файл random_letters.sh, использующий встроенную переменную $RANDOM.

### Листинг программы

#!/bin/bash

ALPHABET=abcdefghijklmnopqrstuvwxyz

for ((i=0; i<10; i++))
do
    INDEX=$((RANDOM % 26))
    echo -n "${ALPHABET:$INDEX:1}"
done

echo

### Результат выполнения

Программа генерирует случайную последовательность букв латинского алфавита.

# Контрольные вопросы

## 1. Найдите синтаксическую ошибку

Исходная строка:

while [$1 != "exit"]

Ошибка заключается в отсутствии пробелов внутри квадратных скобок.

Правильный вариант:

while [ $1 != "exit" ]

## 5. Отличия zsh от bash

Оболочка zsh обладает расширенными возможностями автодополнения и настройки интерфейса. Bash является более стандартной и распространённой оболочкой.

## 7. Сравнение bash с языками программирования

Преимущества bash:
- удобство автоматизации;
- быстрое написание скриптов;

Недостатки:
- неудобство создания крупных программ;
- ограниченные структуры данных;
- низкая производительность по сравнению с компилируемыми языками.
